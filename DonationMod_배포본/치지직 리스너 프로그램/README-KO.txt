CHZZK Donation Listener for Windows (unofficial channel-ID mode)

1. config.json의 네 경로를 실제 PalServer DonationMod Scripts 폴더로 설정합니다.
   리스너가 다른 PC에서 실행된다면, 그 PC가 읽고 쓸 수 있는 네트워크 공유(UNC) 경로를 사용합니다.
2. CHZZK-Donation-Listener.exe를 실행한 뒤 창을 닫지 않습니다.
3. 스트리머는 Palworld 서버에 접속한 뒤 일반 채팅으로 아래처럼 입력합니다.

   !chzzkregister https://chzzk.naver.com/live/채널ID

   또는

   !chzzkregister 채널ID

4. 연결 상태는 !chzzkstatus, 연결 해제는 !chzzkunregister 입니다.

Node.js, Client ID, Client Secret, OAuth 로그인은 필요하지 않습니다.
이 프로그램은 비공식 치지직 채팅 연결을 사용하므로 치지직 내부 변경 시 작동하지 않을 수 있습니다.
